# Dependencies
- [Simple Direct Media Layer 3 (SDL3)](https://github.com/libsdl-org/SDL)
    ```
    sudo pacman -S sdl3
    sudo apt install libsdl3-dev
    brew install sdl3
    ```

- [make](https://github.com/wkusnierczyk/make)
    ```
    sudo apt install make
    sudo pacman -S make
    brew install make 
    ```
